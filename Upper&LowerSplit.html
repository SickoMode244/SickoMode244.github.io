Upper and Lower Body Routine 4 Day Split: (Intermediate)

Monday: Upper Body (1)

Bench Press 3 sets x 6-12, Barbell Row 3 sets x 6-12, Seated Overhead Dumbbell Press 3 sets x 8-12, Pec Dec - 3 sec negative 2 sets x 10-12, V-Bar Lat Pull Down - 3 sec negative 2 sets x 10-12, Side Lateral Raise 2 sets x 10-15, Cable Tricep Extensions - 3 sec negative 3 sets x 8-12, Cable Curls - 3 sec negative 3 sets x 8-12

Tuesday: Lower Body (1)

Squats 3 sets x 6-12, Stiff Leg Deadlifts 3 sets x 8-12, Standing Calf Raise 3 sets x 10-15, Leg Extensions - 3 sec negative 2 sets x 10-12, Leg Curl - 3 sec negative 2 sets x 10-12, Seated Calf Raise - 3 sec negative 2 sets x 10-12, Cable Crunch - 3 sec negative 3 sets x 10-12, Cable Pull Through w/Rope 3 sets x 10-12

Thursday: Upper Body (2)

Incline Dumbbell Bench Press 3 sets x 8-12, Rack Deadlifts - 3" to 5" off ground 3 sets x 5-8, Military Press 3 sets x 8-12, Machine Chest Press - 3 sec negative 2 sets x 8-12, Pull Ups or Machine Rows - 3 sec negative 2 sets x 8-12, Machine Shoulder Press - 3 sec negative 2 sets x 8-12, Dumbbell Curls - 3 sec negative 3 sets x 8-12, Machine Tricep Dip - 3 sec negative 3 sets x 8-12

Friday: Lower Body (2)

Leg Press 3 sets x 10-20, Dumbbell Stiff Leg Deadlifts 3 sets x 8-12, Leg Press Calf Raise 3 sets x 10-15, Hack Squat 2 sets x 8-12, Seated Leg Curl - 3 sec negative 2 sets x 10-12, Seated Calf Raise - 3 sec negative 2 sets x 10-12, Planks 3 sets 60 sec, Hyperextension 3 sets x 10-12

Rest Days:

Sunday, Wednesday, Saturday
